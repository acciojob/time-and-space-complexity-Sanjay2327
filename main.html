Time And Space Complexity

Time complexity measures the amount of time an algorithm takes to run as a function of the length of the input ($n$). It does not measure time in seconds (because hardware speeds vary), but rather the number of elementary operations performed.

Space complexity measures the total amount of memory (RAM) an algorithm uses relative to the input size. This includes both the space needed for the input values and the auxiliary space (temporary memory) used during execution.and O(n) is also good to have .
